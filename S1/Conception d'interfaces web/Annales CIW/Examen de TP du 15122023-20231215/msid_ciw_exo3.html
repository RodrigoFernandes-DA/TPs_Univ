Conception d'Interface Web

Examen pratique de seconde session

Cet examen comprend trois exercices. Pour y répondre, vous devez compléter les fichiers qui vous sont fournis:

  • msid_ciw_exo1.html
  • msid_ciw_exo2.html
  • msid_ciw_exo3.html
  • msid_ciw.css

Les trois exercices sont indépendants et peuvent être traités dans n'importe quel ordre. Pour répondre à chacun d'eux, vous devez compléter le fichier HTML correspondant et/ou le fichier CSS associé.

Exercice 3

Pour cet exercice, vous devez concevoir une galerie d'images responsive dans laquelle les images sont disposées sur plusieurs lignes. Pour cela, vous devez modifier les fichiers msid_ciw_exo3.html et msid_ciw.css.

Par responsive, on entend que toutes les images sont de hauteurs identiques et se redistribuent automatiquement sur les lignes en fonction de la largeur de l'écran. Le résultat attendu est montré dans l'image sample10.jpg de la gallerie qui est entourée en rouge. Pour obtenir ce résultat, vous ne devez pas utiliser de framework (e.g. Bootstrap) mais uniquement les modules CSS standards (e.g. Flexbox).
