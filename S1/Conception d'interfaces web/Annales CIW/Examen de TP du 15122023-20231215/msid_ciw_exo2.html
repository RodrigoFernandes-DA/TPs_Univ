Conception d'Interface Web

Examen pratique de seconde session

Cet examen comprend trois exercices. Pour y répondre, vous devez compléter les fichiers qui vous sont fournis:

  • msid_ciw_exo1.html
  • msid_ciw_exo2.html
  • msid_ciw_exo3.html
  • msid_ciw.css

Les trois exercices sont indépendants et peuvent être traités dans n'importe quel ordre. Pour répondre à chacun d'eux, vous devez compléter le fichier HTML correspondant et/ou le fichier CSS associé.

Exercice 2

Ce deuxième exercice vise à mettre en forme cette page HTML (et uniquement cette page), pour obtenir un résultat le plus proche possible de la figure ci-dessous. Pour cela, vous devez définir des règles de styles CSS dans le fichier msid_ciw.css. Ces règles ne doivent pas impacter la mise en forme des autres pages HTML, même si celles-ci sont aussi associées à cette feuille de style. Si besoin, vous pouvez également modifier le code HTML de cette page.
