Conception d'Interface Web

Examen pratique de seconde session

Cet examen comprend trois exercices. Pour y répondre, vous devez compléter les fichiers qui vous sont fournis:

  • msid_ciw_exo1.html
  • msid_ciw_exo2.html
  • msid_ciw_exo3.html
  • msid_ciw.css

Les trois exercices sont indépendants et peuvent être traités dans n'importe quel ordre. Pour répondre à chacun d'eux, vous devez compléter le fichier HTML correspondant et/ou le fichier CSS associé.

Exercice 1

Pour ce premier exercice, vous devez créer une barre de navigation pour cette page. Cette barre de navigation comprend un logo placé à gauche (logo_blanc.png), et trois items placés à droite:

  • Exercice 1
  • Exercice 2
  • Exercice 3

Chaque item de ce menu de navigation dirige vers l'une des pages de l'énoncé: msid_ciw_exo1.html, msid_ciw_exo2.html et msid_ciw_exo3.html.

Le résultat attendu est montré sur la figure ci-dessous. Pour parvenir à ce résultat, vous devez utiliser le framework Bootstrap et modifier/compléter les fichiers msid_ciw_exo1.html et msid_ciw.css. Sur la figure, le menu Exercice 3 est survolé par le curseur de la souris et le menu Exercice 1 correspond à la page actuellement visitée. Cette barre de navigation est "collée" en haut de la page. Enfin, cette barre de navigation doit être responsive, c'est-à-dire proposer un menu tiroir (toggler) lorsque la largeur de l'écran est trop petite pour afficher le menu complet.
